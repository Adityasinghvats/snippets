VS Code Extension

Snippet Saver

Snippet-saver is a vscode extension tool, which relies on local storage of your device to provide create, read and delete functionality for code snippets. I have made this extension because I don't trust anyone with my code files and want to keep it in my local machine and I believe you should not trust anyone else for your personal data as well.

You can now use @ to get all your saved snippets in the editor.
Download Extension

How to use?

Key Features

Open-Source

This extension is completely open-source for reviews and contributions.

@ search

You can search for your snippets using the @ symbol followed by the snippets title.

Easy to use

Extremely easy to use, just select the code and use the default command-palette.

No Internet

You don't need internet to use this extension, it works offline for privacy reasons.
